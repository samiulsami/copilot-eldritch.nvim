COPILOT ELDRITCH                                             *copilot-eldritch*

A Lovecraftian-themed spinner for Copilot in Neovim. Shows fleeting, changing
eldritch glyphs while Copilot is working in the background.

                                      *copilot-eldritch-installation*
1. Installation ~

Add this as a dependency of `copilot.lua`.

Note: You must call `require("copilot-eldritch").setup()` AFTER
`require("copilot").setup()`.

Using lazy.nvim:
>lua
    {
        "zbirenbaum/copilot.lua",
        dependencies = { "samiulsami/copilot-eldritch.nvim" },
        config = function()
            require("copilot").setup({...})
            require("copilot-eldritch").setup()
        end
    }
<

                                      *copilot-eldritch-configuration*
2. Configuration ~

You can configure the plugin by passing a table to the setup function.

>lua
    require("copilot-eldritch").setup({
        max_length = 73,
        min_length = 1,
        max_distance_from_cursor = 10,
        max_lines = 3,
        repeat_ms = 50,
        jitter_ms = 100,
        rand_hl_group = "CopilotEldritchHLGroup",
        colors = {
            red = { min = 133, max = 255 },
            green = { min = 0, max = 0 },
            blue = { min = 68, max = 68 },
        },
        chars = { ... }
    })
<

OPTIONS                                           *copilot-eldritch-options*

max_length (number) ~
    Maximum length of the generated spell string.
    Default: 73

min_length (number) ~
    Minimum length of the generated spell string.
    Default: 1

max_distance_from_cursor (number) ~
    Maximum random distance from the cursor position to start drawing glyphs.
    Default: 10

max_lines (number) ~
    Maximum number of lines (vertical) to spread the spell across.
    Default: 3

repeat_ms (number) ~
    Base speed of the animation tick in milliseconds.
    Default: 50

jitter_ms (number) ~
    Random jitter added to `repeat_ms` for the flickering disappearance effect.
    Default: 100

rand_hl_group (string) ~
    The highlight group used for the text. By default, the foreground color
    is randomized based on `colors`.
    Default: "CopilotEldritchHLGroup"

colors (table) ~
    RGB ranges for the random text color. Contains `red`, `green`, `blue`,
    each with `min` and `max` (0-255).

chars (table) ~
    A list of strings (glyphs) OR a list of lists of strings.

    If a simple list is provided, characters are chosen randomly from that
    single list.

    If a list of lists is provided (e.g. `{ { "a", "b" }, { "1", "2" } }`),
    one of the inner lists is chosen randomly each time Copilot starts
    loading (InsertEnter/Typing), acting as a "theme" for that specific
    loading sequence.

    Default: A collection of eldritch and unicode glyphs.
